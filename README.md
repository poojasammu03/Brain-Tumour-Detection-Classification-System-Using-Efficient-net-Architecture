# Brain-Tumour-Detection-Classification-System-Using-Efficient-net-Architecture
This project presents an AI-ML based automated system for brain tumour detection and classification from MRI images using a fine-tuned EfficientNet-B0 architecture. The system leverages deep learning and transfer learning to achieve high diagnostic accuracy while maintaining computational efficiency.
